模拟电子电路

第一章 绪论

按时间和幅值的连续性和离散性把信号分为四类：2*2自由组合

输入电阻： 表示了某种放大电路从信号源吸取信号能力的大小

输出电阻： 影响着它带负载的能力，当负载变化时，输出量变化很小或基本不变表示带负载能力强

增益Amplifier： 将供电电源能量转换为信号能量的能力

频率失真： 由线性电抗元件引起的失真

幅度失真： 由于放大电路的带宽所限，导致对不同频率信号幅值的放大倍数不同而产生的失真称为幅度失真

相位失真： 放大电路对不同频率的信号产生的时延不同

第二章 运算放大器

运算放大器的电压传输特性具有两个区域：线性区，饱和区，为使由集成运放所组成的各种应用电路能稳定地工作在线性区，必须引入负反馈，是将运放输出端与反相输入端通过其他元件相连形成负反馈共同构成闭环状态

理想运算放大器：

  • 开环电压增益$A_{vo}\rightarrow\infty$

  • $r_i\rightarrow \infty$

  • $r_o\rightarrow0$

  • 开环带宽$BW\rightarrow\infty$

同相放大电路： 加到两输入端的电压大小接近相等。输入输出相位相同时它在闭环工作状态下的特征。$A_v\ge1,R_i\rightarrow\infty,R_o\rightarrow0$

闭环系统。存在深度负反馈，深度负反馈是虚短的条件

个人理解：同相放大器相当于自控中的闭环比例控制，开环增益为无穷大，导致他的位置误差趋向于0

电压跟随器： 单位反馈的同相放大电路，$R_i\rightarrow\infty,R_o\rightarrow0$

在电路中常用作阻抗变换器或功率放大器，一般接在高阻信号源和低阻负载之间

反相放大电路： 输入输出反向，$R_i = R_1,R_o\rightarrow0$，具有低输入电阻

特征： 存在虚地（同相端接地，由虚短可得）

同相输入和反相输入放大电路的其他应用：

  • 求差电路：

  • 仪用放大电路： 抑制共模信号，抑制干扰信号能力强

  • 求和电路：

  • 积分电路： 先电阻后电容，微分电路相反

第三章 二极管及其基本电路

雪崩击穿： 反向电流极急剧增大，电击穿，可逆

齐纳击穿： 多用于稳压电路，电击穿，可逆

热击穿： 不好，会烧毁二极管，不可逆，尽量避免

二极管： $i_D = I_S(e^{v_D\over v_T}-1),i_D$：流过二极管的电流，$I_S$：反向饱和电流，$v_D$：二极管两端电压，$v_T$ 温度电压当量 硅管死区电压0.5v，锗管0.1v。硅管导通电压0.7v,锗管0.2v

  • 理想模型： 相当于一个开关

  • 恒压降模型： 过了导通电压相当于一个理想开关

  • 折线模型： 开关串联电阻$r_D = 200\Omega$

二极管的应用： 整流电路，限幅与钳位电路，开关电路

特殊二极管：

齐纳二极管：反向击穿，用于稳压

变容二极管：利用结电容，用于高频技术中

肖特基二极管：电容效应小，工作速度快，但反向击穿电压较低

第四章 场效应三极管及其放大电路

场效应管是一种专门利用电场效应来控制其电流大小的半导体器件。这种器件不仅兼有体积小、重量轻、耗电省、寿命长的特点，而且还有输入阻抗高、噪声低、抗辐射能力强和制造工艺简单的优点。

I-V 特性曲线及其特性方程

  1. 输出特性及其特性方程

    MOSFET的输出特性是指在栅源电压v{GS}一定的情况下，漏极电流i_D与漏源电压v{DS}之间的关系，即 iD = f(v{DS})|{v{GS}=常数}

    • 截止区：当v_{GS}

    • 可变电阻区：v{GS}>V{TN}, 其iD=K_n[2(v{GS}-V{TN})v{GS}-v{DS}^2], 直观理解：在可变电阻区，输出特性的斜率随v{GS}变化明显

    • 饱和区（恒流区又称放大区）：v{GS}\ge V{TN}, 可近似看为iD不随v{DS}变化

  2. 转移特性： FET是电压控制器件，所谓转移特性是在漏源电压v{DS}一定的条件下，栅源电压v{GS}对漏极电流i_D的控制特性

N沟道耗尽型MOSFET: 栅源电压称为夹断电压(截止电压)V_{PN}

沟道长度调制效应等几种效应：

  1. 沟道长度调制效应：early电压

  2. 衬底调制效应

  3. 温度效应：MOS管提供了一种反馈条件，自然限制了沟道电流，保证了功率MOS管的稳定运行

  4. 击穿效应

静态工作点求法：

  1. 画出放大电路的直流通路，标出各电流、电压

  2. 计算静态工作点参数I{DQ}=K_n(V{GSQ}-V_{TN})^2

图解分析法： 适用于交流信号幅值较大的情况，可能会产生饱和失真和截止失真

小信号模型分析法：

  1. 求静态值

  2. 求FET的互导和输出电阻

  3. 画出电路的交流小信号等效模型

带源极电阻的共源极放大电路：

  • 源极电阻的作用？稳定静态工作点

  • 副作用？降低电压增益

  • 如何解决？并联旁路电容，交流分析时短路源极电阻

三种放大电路的比较：

结型场效应管（JFET）及其放大电路： 与耗尽型MOSFET相似

  1. JFET栅极，沟道之间的PN结是反向偏置的，因此，其i_G\approx 0,其输入人电阻的阻值很高

  2. JFET是电压控制电流器件，iD受v{GS}控制

  3. 预夹断前，iD与v{DS}呈近似线性关系；预夹断后，i_D趋向于饱和

第五章 双极结型三极管（BJT）及其放大电路

\beta: 共射级电流放大系数

共射级连接时的I-V特性曲线：

  1. 输入特性曲线：描述了当输出电压v{CE}为某一数值时，输入电流i_B与输入电压v{BE}之间的关系，用函数表示为iB=f(v{BE})|{v{CE}=常数}

  2. 输出特性：iC=f(v{CE}|_{i_B=常数})

    • 放大区：发射结正偏电压大于开启电压，而集电结反偏。在放大区域内，BJT输出特性曲线的特点是各条曲线几乎与坐标轴平行，但随着v_{CE}的增加，各条曲线略向上倾斜，这就是基区宽度调制效应。

    • 饱和区：发射结和集电结都正偏，该区域内，v_{CE}

    • 截止区：集电结反偏，发射结上偏置电压小于PN结的开启电压,发射极电流$i_E=0$

饱和失真： Q点过高而产生的失真称为饱和失真，同MOSFET

截止失真： Q点过低，同MOSFET
