自动控制原理与现代控制理论

自动控制原理

第一章 自动控制的一般概念

自动控制：是指在没有人的参与的情况下，利用外加的设备或装置，使机器、设备或生产过程的某个工作状态或参数自动地按照预定的规律运行

反馈控制：取自被控量的反馈信息，用来不断修正被控量与输入量的偏差，从而实现对被控对象进行控制的任务

反馈控制系统的组成：包括被控对象和控制装置。

扰动：是系统不希望的外作用，他破坏有用输入对系统的控制，在实际系统中，扰动可以作用于系统的任何元部件上。

开环控制方式:指控制装置与被控对象之间只有顺向作用而没有反向联系的控制过程，控制精度完全取决于所用元件及校准的精度。开环控制方式没有修正偏差的能力，抗扰性较差。

自动控制系统的分类：

自动控制系统的要求：1. 稳定性 2.快速性 3.准确性

第二章 控制系统的数学模型

控制系统中的数学模型：是描述系统内部物理量之间关系的数学表达式

时域的数学模型：微分方程，差分方程和状态方程

复数域的数学模型:传递函数和结构图

线性系统：可以应用叠加定理，既满足叠加性和均匀性

传递函数：在零初始条件夏，系统的输出量的拉氏变换与输入量的拉氏变换之比，传递函数不仅可以表征系统的动态性能，而且可以用来研究系统的结构或参数变化对系统性能的影响。

传递函数只与系统结构和参数有关。

传递函数的极点：受输入函数激发，在输出响应中形成自由运动的模态。

传递函数的零点：并不形成自由运动的模态，但他们却影响各模态在响应中所占的比重，因而影响响应曲线的形状。

第三章 线性系统的时域分析法

动态系统的评价指标：动态性能和稳态性能

动态过程：指系统在典型输入信号的作用下，系统输出量从初始状态到最终状态的响应过程，通常在阶跃函数的作用下，测定系统的动态性能。

动态性能指标：

  1. 上升时间$t_r$ ：指响应从终值10%上升到中值90%所需的时间

  2. 峰值时间$t_p$：指响应超过其中值到达第一个峰值所需要的时间

  3. 调节时间$t_s$：指响应到达并保持在中值5%内所需要的时间

  4. 超调量：指响应的最大偏离量与终值的差的百分比。

稳态性能指标:

  1. 稳态误差

一阶系统的时域分析

  • 阶跃响应没有误差

  • 斜坡响应存在稳态跟踪误差

二阶系统的时域分析

  • 阻尼比 < 0 系统不稳定

  • 阻尼比 = 0 无阻尼，系统等幅振荡

  • 0 < 阻尼比 <1 欠阻尼，系统震荡衰减

  • 阻尼比 = 1 临界阻尼 阶跃响应非周期的稳态输出

  • 阻尼比 > 1 过阻尼 阶跃响应非周期额稳态输出

注：可以看特征根，或者转换为时域模型帮助理解分析。

二阶系统的性能改善

  1. 比例微分控制：是一种早期控制，可出现在位置误差前，提前产生修正作用，微分系统并不影响系统的稳态误差，但是，它相当于增大了系统的阻尼，从而容许选用更大的开环增益，改善系统的动态性能，不过，微分系统对噪声尤其是高频噪声很敏感，不适用于又高频噪声的系统。

  2. 测速反馈控制：增大系统阻尼，改善系统动态性能。其对输入端信号有滤波作用，测速反馈不会影响自然频率，但会降低系统的开环增益。

线性系统的稳态误差的计算

  1. 在系统稳定的基础山，才能有稳态误差的计算。

  2. $E(s) = R(s) - H(s)C(s)$ 用拉氏终值定理求解

第四章

自动控制原理的概念

补充

最小相位系统：在一定的幅频特性情况下其相移为最小的系统，也称最小相位系统。

第五章

频率特性： 谐波输入下，输出响应中与输入同频率的谐波分量与谐波输入的幅值之比$A(\omega)$为幅频特性，相位之差$\phi(\omega)$为相频特性，并称其指数表达形式$G(j\omega) = A(\omega)e^{j\phi(\omega)}$ 为系统的频率特性。

频率响应： 线性系统稳态正弦相应的幅值，相角随输入频率变化的规律性

典型环节的幅相特性曲线：

频域稳定判据：

  • 系统稳定的充分必要条件：全部的闭环极点均具有负的实部。

  • 代数稳定判据的

    • 由闭环特征多项式系数（不解根）判定系统的稳定性

    • 不能用于研究如何调整系统结构来改善系统的动态性能

  • Nyquist稳定判据： $Z = P - 2N$

    Z：右半平面闭环极点的个数

    P：右半平面开环极点的个数

    N：开环幅相曲线$GG(j\omega)$包围[G]平面(-1,j0)点的圈数，（逆时针为正）

稳定裕度：

  • 相角裕度： 系统在相角上距离临界稳定还具有的储备量

  • 幅值裕度： 系统在增益上距离临界稳定还具有的储备量

三频段理论(只适用于单位反馈的最小相角系统)：

$L(w)—>G(s)—>\Phi(s)—>动态性能$

  • 低频段：系统的稳态误差$e_{ss}$ ,主要有系统型别和开环增益K决定，希望形状：陡，高

  • 中频段：系统的动态性能($\delta\%,t_s = \frac{8.3}{w_c}$)，希望形状：缓，宽

  • 高频段：系统抗高频噪声的干扰能力，希望形状：陡，低

闭环频率特性曲线的绘制：

使用等M圆，等N圆，找点描点

闭环频率特性的几个特征量：

  • 零频值$M_0 = M(0)$

  • 谐振频率$w_r$ ,谐振峰值$M_r$

  • 带宽频率wb：$M(w)$ 下降到$0.707M_0$时对应的频率值

频率法串联校正： 采用适当方式，在系统中加入一些结构和参数可调整的装置（校正装置），用以改善系统的动态性能，使系统满足指标要求。

  1. 超前网络：利用超前网络相角超前特性提高系统的相角裕度$G(s) = \frac{aTS+1}{TS+1},a>1$

  2. 滞后网络：利用滞后网络幅值衰减特性挖掘系统自身的相角储备$G(s) = \frac{1+bTS}{1+TS},0<b<1$

  3. 滞后-超前校正：当超前和滞后校正都不奏效时使用

  4. 串联PID校正：可以提高系统型别，降低稳态误差

线性离散系统的分析与校正

离散系统： 如果控制系统中有一处或几处信号是一串脉冲或数码，换句话说，这些信号仅定义在离散时间上，则这个系统称为离散时间控制系统

离散系统类型：

  • 采样控制系统：时间离散，数值连续

  • 数字控制系统：时间离散，数值离散

    • A/D转换器：把连续的模拟信号转换为离散数字信号的装置

      等效为理想开关($\tau<<T$,字长足够（可以进行比较精确的量化）)，$e^*(t) = e(t)\delta_T(t)$

    • D/A转换器：把离散的数字信号转换为连续模拟信号的装置，用零阶保持器（ZOH）实现

      ZOH：$G_h(s) = \frac{1-e^{-Ts}}{s} \approx e^{-Ts/2}$,类似延迟了二分之一拍的延时环节，会降低相角裕度，是动态性能也降低。

  • Shannon定理：$w_s = \frac{2\pi}{T}\gt 2w_h$ 采样频率要大于信号最高频率的两倍 $T\le\frac{2\pi}{2w_h}$

现代控制理论

控制系统能控能观的物理概念

能控性和能观性是现代控制理论中两个根本性的概念。它们是系统的两个基本结构特性，揭示了动力学系统关于内部的状态与系统输入、输出之间的关系。

能控性指的是控制作用u(t)影响系统状态变量x(t)的能力

能观性指的是由系统的输出y(t)反映系统的状态变量x(t)的能力

前者回答u(t)能否使x(t)作任意转移的问题；后者则回答能否通过y(t)的量来确定x(t)的问题

状态反馈是什么，在工程中有什么应用？

状态反馈：是将系统的每一个状态变量乘以相应的反馈系数，然后反馈到输入端与参考输入相加形成的状态反馈控制律。

输出反馈：是将系统的输出向量乘以相应的反馈系数，然后反馈到输入端与参考输入相加构成输出反馈律。

区别：1、状态反馈是信息系统的一种完全反映；而输出反馈则是系统信息的一部分或不完全的地反映。因而状态反馈能达到的性能，输出反馈则未必能达到。2、状态反馈不改变能控性，可能改变能观性；而输出反馈均不改变能控性和能观性。

第二章 控制系统状态空间表达式的解

系统的自由解： 指系统输入为零时，由初始状态引起的自由运动。线性定常系统的自由解为：$x(t) = e^{At}x_0$

状态转移矩阵： $e^{At}$ 称为状态转移矩阵，从时间的角度而言，这意味着他是状态矢量随着时间的推移，不断地在状态空间中转移。

状态转移矩阵的求法：

  1. 直接法，由级数进行计算

  2. 变换约旦标准型$\Lambda = T^{-1}AT$ ——-> $e^{At} = Te^{\Lambda t}T^{-1}$

  3. 反拉氏变换法

  4. 凯莱-哈密顿定理

线性定常系统非齐次方程的解：

        $x(t) = \Phi(t)x(0) + \int_0^t\Phi(t-\tau)Bu(\tau)d\tau$

  2. 近似离散化：$G = TA + I$

第三章 线性控制系统的能控性和能观性

能控性和能观性正是分析 $u(t)$对状态$x(t)$的控制能力 以及 输出$y(t)$对状态$x(t)$的反映能力

能控性的定义：如果存在一个连续的输入$u(t)$，能在有限的时间区间$[t_0,t_f]$内，是系统的某一初始状态$x(t_0)$，转移到指定的任意终端状态$x(t_f)$，则称此状态是能控的。

能控性的判别方法：

  • 直接从A与B判别系统的能控性

  • 能控判别矩阵：$M = (B,AB,A^2B,…,A^{n-1}B)$，要求M满秩

  • 对于线性连续时变系统，则需要构造格拉姆矩阵进行判别

能观性的定义： 能观性所表示的是输出$y(t)$反映状态矢量$x(t)$的能力，与控制作用没有直接关系。

能观性的判别方法：

  • 直接从A与C判别系统的能控性

  • 能控判别矩阵：$N = (C,CA,CA^2,…,CA^{n-1})^T$，要求N满秩

  • 对于线性连续时变系统，则需要构造格拉姆矩阵进行判别
