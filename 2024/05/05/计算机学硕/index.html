面试问题

OSI 有几层协议， 各叫什么名字？

OSI（Open Systems Interconnection）模型包含七层协议，每一层都有不同的功能和责任。以下是每一层的名称：

  1. 物理层（Physical Layer）
  2. 数据链路层（Data Link Layer）
  3. 网络层（Network Layer）
  4. 传输层（Transport Layer）
  5. 会话层（Session Layer）
  6. 表示层（Presentation Layer）
  7. 应用层（Application Layer）

这些层级协议的目的是为了确保不同计算机系统之间的通信能够有效地进行，并提供了一种标准化的方法来处理数据传输和网络通信。

TCP/IP模型

  1. 网络接口层：物理模型和数据链路

  2. 网络层

  3. 传输层

  4. 应用层

什么是矩阵的秩

矩阵中无关向量组的最大数量，最大无关向量组所含有的向量的个数就是矩阵的秩

矩阵的特征值与特征向量

数据结构里的线性结构有哪些？链表，树，图这些哪些是线性哪些不是？

常见的线性结构：数组，链表，栈，队列，线性表。

数和图不是线性结构，因为节点之间不存在明确的前后关系

什么是信息系统？

信息系统是由计算机硬件、网络和通信设备、计算机软件、信息资源用户组成的以处理信息流为目的的人机一体化系统，实现对信息的输入、存储、处理、输出和控制的功能。

堆和栈的区别

栈的内存由系统自动分配而堆则需要程序员申请。同时栈是一块连续的内存区域，而堆是用链表存储的空闲地址内存，并不连续

heap是堆，stack是栈

stack的空间由操作系统自动分配/释放，heap上的空间手动分配/释放

stack空间有限，heap是很大的自由内存区

程序在编译时对变量和函数分配内存都在栈上进行，且内存运行过程中函数调用时参数的传递在栈上进行

树和图的区别

  1. 树是一种特殊的图，而图不一定是树

  2. 树中不存在回路，而图中一般存在回路

  3. 树是一种非常结构化的数据结构，用于表示层次结构，而图则几乎可以表示任意的关系

求最短路径有哪几种算法，列举一下名称

最短路径：最短路径是指两顶点之间经过的边上权值之和最小的路径

  1. Dijkstra算法：基于贪心策略，逐步确定起始节点到其他节点的最短路径

  2. Floyd算法：通过动态规划的思想，利用矩阵存储节点间的最短距离，逐步更新矩阵以求得最短路径

  3. Bellman-Ford算法：用于解决单源最短路径问题，但相较于Dijkstra算法，Bellman-Ford算法可以处理带有负权边的图。它通过对所有边进行松弛操作的迭代来逐步逼近最短路径

  4. SPFA算法：通过使用队列的方式来避免无效的松弛操作和重复访问节点，从而提高了算法的效率

求最小生成树有哪几种算法，列举下名称

  1. Prim算法（普里姆算法）

  2. Kruscal算法（克鲁斯卡尔算法）

嵌入式系统和嵌入式操作系统的概念

  1. 嵌入式系统：是一种特殊类型的计算机系统，设计用于执行特定的任务或者控制特定的设备，一般集成到其他设备中，比如家电，汽车等

  2. 嵌入式操作系统：嵌入式操作系统是运行在嵌入式系统上的操作系统，它被设计用于管理和控制嵌入式系统的硬件和软件资源，通常满足实时性的要求

什么叫面向对象编程语言？与传统的顺序型编程语言的主要区别是什么?

面向对象编程语言通过封装、继承、多态和类的概念提供了一种更加模块化、可重用和可扩展的编程范式，与传统的顺序型编程语言相比，它更加适合于复杂系统的开发和维护

C++里的虚函数机制是如何实现的？

在C++中，虚函数机制是通过虚函数表（vtable）和虚函数指针（vpointer）来实现的。虚函数允许在基类和派生类之间进行多态的函数调用，即通过基类指针或引用调用派生类的特定实现

C++里的多态机制是如何实现的？

在C++中，多态性是通过虚函数和基类指针或引用实现的。多态性允许通过基类指针或引用调用派生类的特定实现，即在运行时根据对象的实际类型确定要调用的函数

C语言里的指针是指的什么概念？

在C语言中，指针是一种特殊的变量，它存储了一个内存地址，该地址指向存储在计算机内存中的某个特定数据

什么是关系型内存库？

关系型内存库是一种计算机软件，将数据库管理系统和内存运算相结合，它提供了在内存中存储和管理关系型数据的功能。与传统的关系型数据库系统相比，关系型内存库的主要特点是将数据完全存储在内存中，而不是磁盘上。这使得关系型内存库能够提供更快的数据访问和处理速度，适用于对响应时间要求较高的应用场景，实现更高的数据处理性能和更快的查询响应时间

内存具有更快的读写速度和更小的访问延时

什么是关系型数据?

关系型数据是基于关系模型（Relational Model）组织和表示的数据。它是一种结构化的数据模型，使用表格（表）来表示数据之间的关系和联系。关系型数据以行和列的形式存储，并且表格中的每一行代表一个实体或记录，而列则代表实体的属性或字段

Linux(unix)系统与Windows系统的主要区别是？

  1. 内核类型：Linux（Unix）系统使用的是开放源代码的Unix-like内核，如Linux内核，而Windows系统使用的是专有的闭源内核。

  2. 开源与闭源：Linux（Unix）系统是开源的，意味着其源代码可以被用户自由获取、修改和重新分发。而Windows系统是闭源的，其源代码不公开，并且用户无法直接修改和重新分发。

  3. 可定制性：由于开源性质，Linux（Unix）系统具有更高的可定制性。用户可以根据自己的需求自由定制和配置系统。而Windows系统的可定制性相对较低，用户的自定义选项有限。

  4. 文件系统：Linux（Unix）系统使用类Unix的文件系统，如Ext4、XFS等。而Windows系统则使用NTFS（New Technology File System）作为默认文件系统。

  5. 用户界面：Linux（Unix）系统通常提供多种用户界面选择，包括命令行界面（如Bash）和图形用户界面（如GNOME、KDE等），用户可以根据自己的偏好选择。Windows系统主要使用图形用户界面，如Windows桌面环境。
